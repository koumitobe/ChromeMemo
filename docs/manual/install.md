# ChromeMemo インストールマニュアル

> Chrome Web Storeを使わず、ローカルファイルから拡張機能をインストールする手順です。

---

## 前提条件

- Google Chrome がインストールされていること
- 拡張機能のZIPファイル（`ChromeMemo_v1.0.0.zip`）を受け取っていること

---

## Step 1: ZIPファイルを解凍する

受け取った `ChromeMemo_v1.0.0.zip` を任意のフォルダに解凍してください。

> ⚠️ 解凍後のフォルダは**削除しないでください**。拡張機能はこのフォルダを参照し続けます。
> 推奨場所: `C:\Users\ユーザー名\ChromeExtensions\ChromeMemo`（Windows）
> 推奨場所: `~/ChromeExtensions/ChromeMemo`（Mac）

---

## Step 2: Chrome の拡張機能管理画面を開く

1. Chrome のアドレスバーに以下を入力してEnterを押す
   ```
   chrome://extensions
   ```

2. または：右上の「⋮（メニュー）」→「拡張機能」→「拡張機能を管理」

---

## Step 3: デベロッパーモードを有効にする

画面右上の「**デベロッパー モード**」のトグルを **オン** にします。

```
┌─────────────────────────────────────────┐
│  拡張機能              デベロッパーモード ●│  ← ここをオンに
└─────────────────────────────────────────┘
```

---

## Step 4: 解凍したフォルダを読み込む

1. 「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリック
2. 解凍したフォルダ内の **`src`フォルダ** を選択して「フォルダの選択」をクリック

```
ChromeMemo/
└── src/  ← このフォルダを選択
    ├── manifest.json
    ├── background/
    ├── content/
    └── ...
```

---

## Step 5: インストール確認

- 拡張機能一覧に「**ChromeMemo**」が表示されれば成功です
- Chrome のツールバーに 🔵 のアイコンが追加されます
  - 表示されない場合：ツールバー右端の「🧩（拡張機能）」アイコン → ChromeMemo の📌をクリックして固定

---

## ショートカットの確認・変更

アドレスバーに `chrome://extensions/shortcuts` と入力すると、ショートカットを確認・変更できます。

| OS | デフォルトショートカット |
|----|----------------------|
| Mac | `Command + M` |
| Windows / Linux | `Alt + M` |

---

## バージョンアップ手順

1. 新しいZIPファイルを受け取り、**既存フォルダと同じ場所**に解凍（上書き）
2. `chrome://extensions` を開く
3. ChromeMemo の「🔄（更新）」ボタンをクリック

---

## アンインストール手順

1. `chrome://extensions` を開く
2. ChromeMemo の「削除」ボタンをクリック
3. 解凍したフォルダを削除

> ⚠️ アンインストール前にメモをエクスポートしてバックアップを取ることを推奨します。

---

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| アイコンが表示されない | 拡張機能メニュー（🧩）からピン留めする |
| サイドバーが開かない | `chrome://extensions` でChromeMemoが有効になっているか確認 |
| ショートカットが効かない | `chrome://extensions/shortcuts` でショートカットを再設定 |
| メモが消えた | Chromeのデータを消去した可能性あり。定期的にJSONエクスポートを推奨 |
