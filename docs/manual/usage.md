# ChromeMemo 操作マニュアル

---

## 基本操作

### サイドバーを開く / 閉じる

| 方法 | 操作 |
|------|------|
| ショートカット（Mac） | `Command + M` |
| ショートカット（Windows） | `Alt + M` |
| ツールバーアイコン | 🔵 ChromeMemo アイコンをクリック |

---

### メモを作成する

1. サイドバーを開く
2. **「＋ 新規メモ」** ボタンをクリック
3. タイトルと本文を入力
4. **「保存」** ボタンをクリック

---

### メモを編集する

1. メモ一覧から編集したいメモをクリック
2. タイトル・本文を編集
3. **「保存」** ボタンをクリック（保存しないと変更は失われます）

---

### メモを削除する

1. 削除したいメモを開く
2. **「削除」** ボタンをクリック
3. 確認ダイアログで **「OK」** をクリック

---

## 検索・フィルタ

### キーワード検索

サイドバー上部の **検索ボックス** にキーワードを入力すると、タイトル・本文・タグが即時フィルタリングされます。

### タグで絞り込む

- タグボタンをクリックするとそのタグのメモだけ表示されます
- **「すべて」** をクリックするとフィルタを解除します

---

## タグ機能

### タグを追加する

1. メモを開いた状態でタグ入力欄をクリック
2. タグ名を入力（`#` は不要）
3. `Enter` または `,` を押して確定

### タグを削除する

タグチップの **「×」** をクリックします。

---

## ピン留め

重要なメモを一覧の上部に固定できます。

- メモ一覧で右上の **📌 アイコン** をクリックするとピン留め
- もう一度クリックするとピン解除

---

## Markdownプレビュー

本文欄に **Markdown記法** で書いたテキストをプレビューできます。

**「👁 プレビュー」** ボタンをクリックして切り替えます。

### 対応記法

| 記法 | 表示 |
|------|------|
| `# 見出し1` | 大きな見出し |
| `## 見出し2` | 中くらいの見出し |
| `### 見出し3` | 小さな見出し |
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `- リスト項目` | 箇条書き |
| `` `コード` `` | インラインコード |
| ` ```コードブロック``` ` | コードブロック |
| `[リンク](https://...)` | リンク |
| `---` | 水平線 |

---

## URLの挿入

メモ編集中に **「🔗 URL挿入」** ボタンをクリックすると、現在閲覧中のページのURLがカーソル位置に挿入されます。

---

## データのバックアップ・移行

### エクスポート（バックアップ）

1. サイドバー右上の **⚙️（設定）** をクリック
2. **「📤 メモをエクスポート（JSON）」** をクリック
3. `chromememo-backup-YYYYMMDD.json` がダウンロードされます

> 💡 定期的にエクスポートしてバックアップを取ることを推奨します。

### インポート（復元・移行）

1. **「📥 メモをインポート（JSON）」** をクリック
2. エクスポートしたJSONファイルを選択
3. 同じIDのメモは上書き確認ダイアログが表示されます

---

## サイドバーのリサイズ

サイドバー左端の **境界線をドラッグ** することで幅を変更できます。

- 最小幅: 240px
- 最大幅: 600px

---

## データについて

- メモはブラウザのローカルストレージに保存されます
- **Chromeの「閲覧データを消去」を実行するとメモも削除されます**（事前にエクスポートを推奨）
- 別のPCへ移行する場合はエクスポート → インポートを使用してください
